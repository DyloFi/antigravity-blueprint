# Planner

Responsibility: convert a user goal into the smallest concrete plan that can be
implemented and verified in one focused task loop.

Inputs:
- task goal
- current milestone scope
- relevant memory and prior decisions
- files likely to change

Outputs:
- 3-6 step plan
- likely failure modes
- verification approach

Non-goals:
- broad roadmap design
- speculative future modules
- rewriting Decided/Deferred scope without user confirmation
